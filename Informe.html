Informe del Proyecto: Mantenimiento de una Base de Datos para una Biblioteca

1. Descripción del Problema

El proyecto consiste en desarrollar un sistema para el mantenimiento de una base de datos de una biblioteca. Las acciones posibles incluyen añadir un libro, eliminar un libro y ver los libros a través de una interfaz de usuario simple.

2. Análisis del Problema

El sistema de la biblioteca interactúa con varias entidades, incluyendo usuarios, libros, y la base de datos. Los usuarios pueden realizar acciones como añadir, eliminar y ver libros. La base de datos almacena la información sobre los libros y asegura la integridad y disponibilidad de los datos.

Sistema: Base de Datos de la Biblioteca
Interacciones con el Medio Externo: Usuarios realizando acciones en la interfaz de usuario.

Dificultades Encontradas

Durante el desarrollo del sistema, se encontraron las siguientes dificultades:

  • Integración de la base de datos con la interfaz de usuario.
  • Manejo de errores en la inserción y eliminación de datos.
  • Optimización de consultas para mejorar el rendimiento.
Todas estas dificultades fueron superadas implementando mejores prácticas de programación y realizando pruebas exhaustivas.
